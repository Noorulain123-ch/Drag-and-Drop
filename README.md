#  Simple Kanban Board

A clean and interactive **Kanban Board** built using HTML, CSS, and JavaScript. This application helps users manage tasks visually by organizing them into different stages.


##  Features

*  Task organization into three stages:

  * To Do
  * In Process
  * Done
*  Drag and drop tasks between columns
*  Simple and user-friendly interface
*  Real-time task movement


##  Technologies Used

* HTML
* CSS
* JavaScript


##  How It Works

* Tasks are displayed in three columns: **To Do**, **In Process**, and **Done**
* Drag a task card from one column
* Drop it into another column
* The task updates its position instantly
